# 家庭成长激励助手 V1.0 设计文档

## 1. 项目概述
- 产品名称：家庭成长激励助手（Family Rewards）
- 版本：V1.0（MVP）
- 技术栈：Expo SDK 51 + React Native + Tamagui + Expo Router + Zustand + SQLite
- 核心定位：单机双模式家庭教育工具，通过"任务-积分-奖励"正向循环培养孩子良好习惯
- 目标用户：家长（管理端）、孩子（使用端）

## 2. 技术架构设计
采用经典分层架构，各层职责明确，便于维护和扩展：

### 2.1 分层结构
```
UI层（Tamagui组件 + Expo Router页面） → 状态层（Zustand） → 服务层（业务逻辑封装） → 数据层（SQLite + Expo Secure Store）
```

### 2.2 目录结构
```
app/                    # Expo Router 路由目录
  (parent)/             # 家长模式路由组
  (child)/              # 孩子模式路由组
src/
  components/           # 通用组件库
  store/                # Zustand状态管理（按业务拆分切片）
  services/             # 服务层（数据库操作、Expo API封装）
  utils/                # 工具函数
  themes/               # Tamagui主题配置
  types/                # TypeScript类型定义
```

## 3. 数据库设计
### 3.1 核心表结构
| 表名 | 核心字段 | 用途 |
|------|----------|------|
| children | id, name, avatar, theme_color | 孩子信息（支持多孩子） |
| tasks | id, name, icon, point, type, category, require_photo, auto_approve | 任务库 |
| task_submissions | id, task_id, child_id, status, photo, submit_time | 任务提交审核记录 |
| points_log | id, child_id, change, reason, type, created_at | 积分流水账 |
| wishes | id, name, image, target_point, type, status | 愿望清单 |
| badges | id, name, icon, condition_type, condition_value | 勋章库 |
| user_badges | id, child_id, badge_id, obtained_at | 孩子获得的勋章记录 |
| settings | key, value | 系统配置（密码、密保、设置等） |

## 4. 核心功能模块设计

### 4.1 公共功能
- **模式切换**：家长→孩子一键切换，孩子→家长需4位PIN码/生物识别验证
- **安全机制**：密码加密存储，连续3次错误锁定60秒，密保问题找回
- **数据备份**：支持本地导出/导入SQLite备份文件

### 4.2 家长模式功能
1. **首页控制台**：孩子状态看板、待审核任务流、快捷奖惩悬浮按钮
2. **任务管理**：内置模板库、自定义任务、任务启停、审核流程
3. **愿望管理**：愿望添加/编辑/锁定、图片存储、进度跟踪
4. **孩子管理**：多孩子添加/切换、专属主题色设置、数据清理
5. **统计与勋章**：积分流水统计、勋章管理、成长报表
6. **系统设置**：安全设置、数据管理、会员管理、通用设置

### 4.3 孩子模式功能
1. **我的主场**：积分展示、主攻目标进度、今日成就
2. **任务大厅**：瀑布流任务展示、任务提交、审核状态反馈
3. **梦想宝库**：愿望列表、兑换流程、历史奖励记录
4. **荣誉与协作**：勋章架、成就解锁、家庭共同目标

## 5. UI/UX设计
### 5.1 主题系统
- 家长模式：简约蓝/绿色调，白底卡片，高效操作导向
- 孩子模式：活泼多彩渐变背景，大图标大按钮，游戏化风格
- 支持按孩子专属主题色动态切换UI色调

### 5.2 动效与音效
| 场景 | 音效 | 动效 |
|------|------|------|
| 积分增加 | 金币叮当声 | 金币飞行动效 |
| 积分扣除 | 沉闷嗡声 | 存钱罐抖动效果 |
| 愿望兑换 | 礼花欢呼声 | 全屏烟花特效 |
| 勋章解锁 | 轻快提示音 | 光芒渐变动画 |
| 模式切换 | 过渡音效 | 变身动画（≤500ms） |

## 6. 开发计划概览
总开发周期约6-8周，分为：
1. 基础架构搭建（1周）
2. 公共模块开发（1周）
3. 家长模式功能开发（2周）
4. 孩子模式功能开发（2周）
5. 测试与优化（1-2周）
