## 三、技术方案

### 3.1 状态管理
使用 React useState 管理表单状态：
- 任务表单：taskName, taskPoint, taskType, taskCategory, requirePhoto, autoApprove
- 愿望表单：wishName, wishPoint, wishType, isMainGoal

### 3.2 API 调用
- 任务创建：taskService.create()
- 愿望创建：wishService.create()
- 愿望兑换：wishService.exchange()

### 3.3 UI 组件
- 使用 Tamagui 组件库
- 表单使用 Input、Button、Switch、XStack、YStack 等组件
- 使用 Dialog 或 Tab 切换显示表单

## 五、预期结果

### 5.1 任务管理页面
- 用户可以查看任务列表（显示任务名称、类型、分类、分值）
- 用户可以创建新任务（设置名称、分值、类型、分类、是否需要拍照、是否自动通过）

### 5.2 愿望管理页面
